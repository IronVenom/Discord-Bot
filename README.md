A Discord Bot made using Python and Discordpy as a pet project. It's a complete server bot.

* Safygiphy
* Wikipedia
* NewsAPI

* Lyricwikia

* XKCD JSON API
* Exchangeratesapi.io	
* Robohash API
* NASA API
